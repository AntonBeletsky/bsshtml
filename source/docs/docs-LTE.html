Bootstrap Lux Documentation

A comprehensive guide to Bootstrap components styled with the Lux theme, featuring an AdminLTE-inspired layout.

Code released under the MIT License by Thomas Park. Based on Bootstrap with icons from Bootstrap Icons.

Introduction

Welcome to Bootstrap with the Lux theme, a powerful front-end framework for building responsive, mobile-first websites.

Example

Hello, world!

Welcome to Bootstrap with the Lux theme.

Code

<div class="container">
  <h1>Hello, world!</h1>
  <p>Welcome to Bootstrap with the Lux theme.</p>
  <button class="btn btn-primary">Get started</button>
</div>

Explanation

Bootstrap is a powerful front-end framework with key features:

  • container: Centers content.
  • btn btn-primary: Styles interactive buttons.
  • Responsive grid and components for mobile-first design.

Use Bootstrap to quickly create modern, responsive layouts.

Accessibility

Bootstrap provides a foundation for accessible websites. Developers are responsible for full accessibility.

Example

Code

<div class="container">
  <button class="btn btn-primary" aria-label="Submit form">Submit</button>
  <img src="https://via.placeholder.com/150" alt="Placeholder image" class="img-fluid rounded">
  <button type="button" class="btn btn-primary">
    <i class="bi bi-search"></i>
    <span class="visually-hidden">Search</span>
  </button>
</div>

Explanation

Bootstrap includes accessibility features:

  • aria-label: Provides screen reader descriptions.
  • alt: Describes images for screen readers.
  • visually-hidden: Hides content visually but keeps it accessible.
  • High-contrast colors and keyboard navigation support.

Ensure components use proper ARIA attributes and semantic HTML.

CSS Variables

Bootstrap leverages CSS variables for theme flexibility.

Example

Code

  --bs-primary: #007bff;
}
<button class="btn btn-primary">Custom Button</button>

Explanation

Key concepts:

  • :root variables (e.g., --bs-primary): Define theme colors.
  • Overwriting variables changes component styles site-wide.
  • Works alongside Sass for comprehensive customization.

Modify CSS variables to adjust Bootstrap’s colors and behaviors at runtime.

Layout

The layout defines the structure of a webpage using semantic HTML and Bootstrap's grid system.

Breakpoints

Breakpoints are predefined screen sizes for responsive design.

Example

Responsive

Code

<div class="container">
  <div class="row">
    <div class="col-12 col-sm-6 col-md-4 col-lg-3 col-xl-2 col-xxl-1 bg-primary text-white p-2">Responsive</div>
  </div>
</div>

Explanation

Key classes:

  • col-*: Applies at all sizes (≥0px).
  • col-sm-*: Small screens (≥576px).
  • col-md-*: Medium screens (≥768px).
  • col-lg-*: Large screens (≥992px).
  • col-xl-*: Extra large (≥1200px).
  • col-xxl-*: Extra extra large (≥1400px).

Z-index

Z-index controls element stacking order.

Example

z-index: 10
z-index: 5

Code

<div class="container">
  <div class="position-relative">
    <div class="position-absolute bg-primary text-white p-2" style="z-index: 10;">z-index: 10</div>
    <div class="position-absolute bg-secondary text-white p-2" style="z-index: 5; left: 20px; top: 20px;">z-index: 5</div>
  </div>
</div>

Explanation

Key classes and properties:

  • z-index: Sets stacking level (e.g., 5, 10).
  • position-*: Required for z-index (e.g., position-absolute).
  • Utility classes: z-0, z-1, etc., for predefined levels.

CSS Grid

CSS Grid provides a two-dimensional layout system.

Example

2fr

Code

  </div>
</div>

Explanation

Key classes and properties:

  • d-grid: Enables CSS Grid.
  • gap-*: Sets spacing between grid items.
  • grid-template-columns: Defines column sizes (e.g., 1fr 2fr).
